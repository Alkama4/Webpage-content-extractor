# Vue.js README

Open frontend directory

```sh
cd .\frontend\
```

Download node_modules

```sh
npm i
```

Compile and Hot-Reload for Development

```sh
npm run dev
```

Type-Check, Compile and Minify for Production

```sh
npm run build
```

## Quick start

Quickly startup the server with one command

```sh
cd .\frontend\
npm run dev
```
